Privacy Policy — Creator OS

Last updated: July 2026

Creator OS is a personal, single-user content-creation dashboard built and used by one individual creator to track their own social media accounts. It is not a public product, is not distributed to other users, and does not accept sign-ups from anyone other than its one developer/operator.

What data this app accesses

When the operator connects a social account (TikTok, YouTube, Twitch, or Instagram) through that platform's official OAuth login, this app requests only basic public profile information and account statistics (such as follower count, follower/subscriber count, view count, and live status) belonging to that same operator's own account.

How data is stored

Access tokens and account statistics are stored locally on the operator's own computer, in a local file that is never uploaded to any server, database, or third party. This application has no shared backend, no user accounts beyond the operator, and no analytics or tracking of any kind.

Data sharing

No data collected by this application is sold, shared, rented, or disclosed to any third party for any purpose. Data is used exclusively to display the operator's own account metrics back to the operator, on the operator's own device.

Data deletion

The operator can disconnect any connected account at any time from within the app, which immediately deletes the stored access token for that account from local storage.
